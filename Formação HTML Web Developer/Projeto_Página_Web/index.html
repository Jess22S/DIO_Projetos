Comunicação Cliente x Servidor

História da Web

  1. Início da Criação da Internet

    2. Durante a Guerra Fria, foi necessário arrumar uma forma de proteger as informações e comunicações das pesquisas americanas dos soviéticos, por isso começaram as pesquisas e o desenvolvimento tecnológico com o objetivo de encontrar um meio de comunicação seguro e que sobrevivesse a um ataque nuclear.

    Em 1965 conseguiram realizar uma conexão bem sucedida de 2 computadores distantes com o conceito de uma rede galáctica para acessar rapidamente dados de qualquer lugar do mundo, mas o nascimento da Internet foi considerado ao criar a ARPANET, no qual, governo americano permitiu que as pesquisas fossem realizadas não só pelos militares como também pelas universidades.

    Em 1969, houve uma conexão realizada com sucesso entre a Universidade da Califórnia e a de Stanford e a primeira queda na Conexão, meses depois 4 universidades americanas estavam conectadas. E a partir daí, deu-se início ao uso e ao aprimoramento da Internet.

  2. Surgimento da Web

    3. Em 1989, Tim Berners-Lee criou a World Wide Web, ou seja, o tão famoso www que tanto utilizamos hoje! Este sistema foi criado para conectar as universidades com o intuito de fazer com que os trabalhos e as pesquisas acadêmicas fossem utilizados ao mesmo tempo. Este cientista também é responsável pelo desenvolvimento de duas ferramentas indispensáveis para a Internet: o Código HTML e o Protocolo HTTP.

    Muitas vezes, Internet e Web parecem ter o mesmo significado, mas não têm, elas tem características diferentes, pois a Internet é uma rede que conecta milhões de computadores pelo mundo, enquanto a Web é uma das ferramentas utilizadas para acessar a Internet.

  3. World Wide Web (www)

    4. Funciona como um Sistema de Sistribuição de Documentos de Hipertexto (HTTP) interconectados e acessíveis por meio de um Navegador Web conectado à Internet. Com o www, conseguimos acessar a Internet nesse formato que usamos hoje. Antes da Web e dos Navegador Web, a forma de acessar a Internet era por meio do Unix.

O que são Clientes?

Cliente é o Computador que está do lado do Usuário e que foi configurado para utilizar o Serviço que é oferecido pelo Servidor, ou seja, o Computador Cliente é aquele que se ocupa de um Serviço.

Início


O que são Servidores?

Servidor é o Computador no qual foi configurado em uma Rede para oferecer algum tipo de Serviço, ou seja, o Computador Servidor é aquele que oferece um Serviço ao Usuário.

Início


O que é HTML?

HTML (HyperText Markup Language ou "Linguagem de Marcação de Hipertexto") é uma Linguagem de Marcação utilizada na construção de Páginas na Web que é interpretada pelos Navegadores Web. A tecnologia é fruto da junção entre os padrões HyTime e SGML.
  • HyTime: É um padrão para a Representação Estruturada de Hipermídia e conteúdo baseado em Tempo, ou seja, é visto como um Conjunto de Eventos Concorrentes dependentes de Tempo (como áudio, vídeo, etc.), conectados por Hiperligações (em inglês: Hyperlink e Link);
  • SGML: É um padrão de formatação de textos que NÃO foi desenvolvido para Hipertexto, mas tornou-se conveniente para Transformar Documentos em Hiper-objetos e para Descrever as Ligações.

Introdução ao HTML na Prática

O que é e como funciona o Inspetor de Elementos?

  1. DevTools: São Conjuntos de Ferramentas de Desenvolvimento integradas aos Navegadores modernos (como Chrome, Firefox, Edge e Safari). Cada Navegador apresenta seu próprio Painel de DevTools, mas a ideia central é a mesma:

    1. Fornecer recursos para Inspecionar;
    2. Editar e/ou Analisar a Página sem que seja necessário sair do próprio Navegador.

  2. Importância do DevTools:

    1. Visualizar a Estrutura do DOM e suas Alterações em Tempo Real;
    2. Alterar Propriedades e Atributos HTML para Testar Comportamentos;
    3. Analisar Problemas e Debugar Possíveis Erros;
    4. Compreender o Funcionamento das Páginas;
    5. Realizar Ajustes Rápidos.

  3. Acesso ao DevTools:

    1. Browser Chrome (ou Edge Chromium): Clique com o botão direito em qualquer local da página e selecione “Inspecionar elemento” ou use o Atalho F12;
    2. Firefox: Clique com o botão direito e escolha “Inspecionar” ou use Ctrl + Shift + C (no Windows) / Cmd + Opt + C (no Mac). Navegando pelo Painel de Elementos.

  4. Benefícios da DevTools: A visualização da Aba Elementos (Chrome) ou do Inspector (Firefox) tem como vantagem:

    1. Mostrar a Árvore de Elementos do HTML (DOM) em Tempo Real;
    2. Identificar a Tag onde determinado Conteúdo ou Componente está localizado;
    3. Verificar quais são os Atributos utilizados (como id e/ou class)
    4. Mostrar a Estrutura Hierárquica, pois fica mais fácil de entender como as Tags estão Aninhadas;
    5. O Código HTML pode ser editado diretamento do Navegador através da Aba Elementos..

Como é a Estrutura Básica do HTML?

A Estrutura Básica do HTML é composta por algumas Tags importantes, mas qual é a função de cada uma delas?
  1. <!DOCTYPE html> : Informa ao Navegador que esse Documento é do Tipo HTML e indica sua Versão;
  2. <html> : É a Raiz do Documento, serve com um Container que engloba TODOS os outros Elementos HTML;
  3. <head> : Compreende as Informações do Documento que serão Interpretadas pelo Navegador (Metadados), como, por exemplo, o <title>;
  4. <meta> : Define Metadados, ou seja, Informações sobre Dados de um Documento HTML, como, por exemplo, o Autor ou a Autora do Documento
  5. <style> : Essa tag é usada para declarar estilos (CSS) para um documento.
  6. <body> : É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário ou usuária vê e interage, nele os conteúdos são estruturados pelas demais tags do HTML;
  7. <link> : É uma tag vazia, que contém apenas atributos e faz a relação do documento HTML com recursos externos. É comumente usado para vincular uma folha de estilo externa, também é usada para definir o favicon da página (ícone da aba do Navegador), como outros recursos;

O que são e como funcionam as Tags?

O que são Tags HTML?
  • São Elementos que Marcam e Formatam os Conteúdos Textuais ou Visuais de Documentos HTML.
  • Elas possuem como Características um Nome e uma Sintaxe que são Específicos de cada Tag.
  • Para Finalizar a Tag, usa-se uma Barra (/), mas há 2 Tipos de Estrutura da Tag:

    1. Possui 2 Tags da Mesma: Uma Tag de Abertura Sem a Barra e uma de Fechamento Com a Barra. Exemplo: <li> </li>;
    2. Possui Somente 1 Tag: A Tag Única já possui a Barra de Fechamento. Exemplo: <br/>.
Como funcionam as Tags HTML?
  • Elas definem os Conteúdos dos Documentos HTML.
  • As Tags vão Mostrar ao Navegador os Conteúdos que deverão ser Interpretados.
  • Elas possuem a seguinte Estrutura:

    1. Tag de Abertura Sem a Barra: Menor que - Conteúdo - Maior que. Exemplo: <p>;
    2. Tag de Fechamento Com a Barra: Menor que - Barra - Conteúdo - Maior que. Exemplo: </p>;
    3. Tag Única Com a Barra de Fechamento: Menor que - Conteúdo - Barra - Maior que. Exemplo: <br/>.

Quais são as Tags mais utilizadas?

Tags Estruturais
  1. <header> </header>: Diferente da Tag <head>, ela determina o começo de Todo o Código e tem a função de Estruturar Todo o Texto e seu trabalho como um todo;
  2. <main> </main>: Define o Conteúdo Central da Página, ainda que este não esteja localizado de fato no centro;
  3. <footer> </footer>: Define o Rodapé da Página, ou seja, são as Informações Finais da Página, pode colocar qualquer tipo de informação, mas geralmente é colocado a Data, Informações de Direito Reservado e Redes Sociais;
  4. <section> </section>: Cria Seções na Página;
  5. <article> </article>: Para quem tem uma Página mais voltada para Conteúdos, ou mesmo um Blog, a Tag <article> é muito importante para a Separação de Informações para que o Artigo possa se destacar;
  6. <aside> </aside>: São Tags que relacionam Conteúdos de uma Seção com algum Conteúdo da Página, ou seja, ela organiza para que as Informações fiquem bem disponibilizadas;
  7. <nav> </nav>: Criar Menus ou ter Informações dispostas de Maneira Organizada;
  8. <div> </div>: Serve para Separar os Conteúdos, antigamente era colocada com um Atributo "class" ou "id" para identificar se era um Rodapé ou um Cabeçalho, por exemplo: <div class="cabecalho"> </div>:, mas como tinha muitas variações nos nomes das Classes, foram criadas Tags Espacíficas para cada parte da Página, mas a Tag <div> </div> ainda é muito utilizada para auxiliar Comandos CSS em Partes Específicas da Página HTML.
Tags de Conteúdo
  1. Tags de Título: São as Tags que mudam o Tamanho do Título, são as Tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6> ;
  2. Tags de Texto: São as Tags que vão "Pular a Linha e Alinhar o Texto" ou Destacar um Texto ou Palavra;
  3. Tags de Link: Essa Tag pode Linkar Qualquer Coisa, dentro da mesma página, ou mesmo alguns elementos que não são da mesma página, é utilizada a Tag <a href="..."> </a>;
  4. Tags de Multimídia: É utilizada para Incorporar Imagens, Vídeos, Sons, Iframes, e qualquer outro tipo de multimídia, para Imagens, por exemplo, é utilizada a Tag <img> </img>, sem fechamento;
  5. Tags de Listas: Organiza informações que são pensadas para Determinar uma Ordem, podendo ser Numérica (<ol> </ol>) ou Não (<ul> </ul>), mas ambas precisam da Tag <li> </li> para Listar os Itens;
  6. Tags de Formulário: As Tags de Formulário (<form> </form>) são indicadas para colher Informações dos Usuários por meio de Dados Simples que são colocados na Tag <input> </input>, e são definidos pelo Tipo (type) de Informação a ser colhida (Nome - name, Email ou Senha - password, por exemplo), ou na Tag <textarea> </textarea> para Informações do Tipo Mensagens, Opiniões ou Reclamações.
Tags de Texto
  1. <p> </p>: Define o Início do Parágrafo;
  2. <span> </span>: É utilizada em elementos como Imagens, Legendas ou Informações que vão somar ao texto, formando uma espécie de Container;
  3. <pre> </pre>: Utilizada principalmente para a utilização de Textos Pré Formatados, como Códigos, por exemplo;
  4. <b> </b>: Transforma Todo o Texto dentro desse intervalo em Negrito;
  5. <i> </i>: Transforma Todo o Texto dentro desse intervalo em Itálico;
  6. <br/>: Tag de HTML utilizada para Quebra de Linha e que não necessita de fechamento;
  7. <hr/>: Formação de Linha Horizontal no texto;
Tags de Estilo
  1. <style> </style>: Deve ser posicionada após a Tag <head> </head> e serve para colocar os Comandos CSS;
  2. <script> </script>: Pode ser posicionada em qualquer local, sendo recomendada após o <footer> </footer> por conta da linguagem, e serve para colocar os Comandos do JavaScript.

O que são Atributos das Tags?

  • Atributos são utilizados para personalizar as Tags, e vão dar todas as Informações que de fato são indicadas..
  • Os Atributos mais utilizados são:

    1. class=”…“: É a Classe de um Elemento, podendo ser utilizada em um ou mais Elementos;
    2. id=”…“: É o Id de um Elemento, podendo ser utilizada somente em um Elemento;
    3. style=”…”: Inclusão de Elementos CSS, que vão dar Estilos ao HTML;
    4. lang=”…”: É o Idioma Principal;
    5. title=”…”: É o Título do Elemento;
    6. alt=”…”: Geralmente, é aplicado em Técnicas de SEO para criar Textos Alternativos nas Imagens;
    7. hidden: Utilizado para ocultar Elementos;
    8. align=”…”: Utilizado para alinhar Textos e/ou Outros Elementos;
    9. width=”…”: Define a Largura do Elemento;
    10. height=”…”: Define a Altura do Elemento.
